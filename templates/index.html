About This App

This Facial Emotion Detection App uses a deep learning model trained on the FER2013 dataset. The dataset contains 35,887 grayscale images of 48x48 pixels, each labeled with one of seven emotions.

Features:

  • Upload an image and detect emotion instantly
  • Model trained on FER2013 dataset with high accuracy
  • Visual display of uploaded image and prediction
  • Confidence score for prediction

How It Works

  1. Select Image: Upload a clear facial image (grayscale preferred).
  2. Model Prediction: The image is processed and passed through a trained neural network.
  3. Emotion Classification: The model classifies the emotion from 7 categories.
  4. Result Display: Predicted emotion and confidence score are shown on screen.
